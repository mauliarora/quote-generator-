 Quote Generator

A modern and interactive Quote Generator web app built using HTML, CSS, and JavaScript.
It fetches random quotes from an API and displays them with a clean glassmorphism UI.

 🚀 Features

* 🎲 Generate random quotes
* ✍️ Displays author name
* 🎨 Modern glass UI design
* ⚡ Smooth animations
* 🌐 API integration

 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* API (DummyJSON / Quotable)

 ▶️ How to Run

1. Download or clone the repo
2. Open `index.html` in browser
3. Click **New Quote**
